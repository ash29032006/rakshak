# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of RAKSHAK seriously. If you find a vulnerability, please report it privately to the maintainer.

As this is a safety application, privacy and security are our top priorities. All detection happens on-device and no audio or video data is ever transmitted to the cloud.
